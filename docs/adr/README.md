## Creating an ADR

Copy `adr-template.md` and fill in the sections.
Name the file: `adr-NNN-short-title.md` (e.g., `adr-001-authentication-planes.md`).

*ADRs are required before the first @E packet. See ADOPTION-ASSESSMENT.md for which ADRs are flagged.*
